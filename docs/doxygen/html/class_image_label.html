Detailed Description

The CameraViewLabel class is a QLabel subclass that displays a resizable pixmap while maintaining aspect ratio.

The CameraViewLabel class provides a QLabel with additional functionality to display a pixmap while preserving its aspect ratio. It automatically scales the pixmap to fit within the label's size, maintaining the original aspect ratio of the pixmap. It also overrides certain event handling methods to customize the behavior of the label.
