◆ startCameraThread()

ThreadResult SickGUI::startCameraThread ( )
private

Starts camera thread.

Creates and open a Camera, assigns camera to a CaptureThread, and starts the CaptureThread. Also connects the CaptureThread's newFrameset signal to SickGUI::newFrameset slot.

Returns
true if started, false otherwise.

◆ startPlcThread()

ThreadResult SickGUI::startPlcThread ( )
private

Starts plc thread.

Creates snap7 client, connects to plc, and starts PlcThread. Also connects the CaptureThread's newFrameset signal to PlcThread::newFrameset slot.

Note
This method cannot be called unless startCameraThread() was called and returned true.
Returns
true if started, false otherwise.
