Creates a "draft" instance of ConnectionGraphicsObject.

The scene caches a "draft" connection which has one loose end. After attachment the "draft" instance is deleted and instead a normal "full" connection is created. Function

Creates Node and Connection graphics objects.

Function is used to populate an empty scene in the constructor. We perform depth-first AbstractGraphModel traversal. The connections are created by checking non-empty node Out ports.
