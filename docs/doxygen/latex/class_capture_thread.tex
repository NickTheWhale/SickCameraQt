\doxysubsection{Detailed Description}
The \doxylink{class_capture_thread}{Capture\+Thread} class handles the capture of frames from a camera. 

The \doxylink{class_capture_thread}{Capture\+Thread} class is a QThread subclass that captures frames from a camera and emits signals whenever a new frameset is available or the connection to the camera is lost.
