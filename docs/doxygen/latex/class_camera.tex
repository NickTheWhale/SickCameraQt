\doxysubsection{Detailed Description}
The \doxylink{class_camera}{Camera} class represents a camera device. 

The \doxylink{class_camera}{Camera} class provides an interface for interacting with camera devices. It defines a set of pure virtual functions that must be implemented by derived classes to perform camera-\/specific operations such as opening, closing, starting and stopping capture, retrieving framesets, and getting camera parameters.
